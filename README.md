## Install & Run*

    wget "url to our tarball"
    tar -xvf projectZer0.tar.gz
    cd zer0
    conda create zer0
    conda activate zer0
    python manage.py runserver
    
   *will run locally and be accessable at localhost:8000 or 127.0.0.1:8000.
